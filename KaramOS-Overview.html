🌐 التطبيق المباشر 💻 مستودع GitHub
ما هو المشروع؟

Karam OS هو لوحة تحكّم شخصية تنظّم عليها مهامك ومشاريعك في مكان واحد. تقدر تضيف مهام بأولويات وتواريخ استحقاق وحالات، تربطها بمشاريع، وتشوف "مهام اليوم" وعرض أسبوعي على شكل تقويم، ونسبة إنجاز لكل مشروع. كل مستخدم يشوف بياناته فقط بفضل نظام حماية على مستوى الصفوف (RLS)، ويوصله ملخّص يومي بالإيميل بمهام اليوم.

ما الذي تم إنجازه

🔐 مصادقة آمنة

تسجيل دخول بالبريد وكلمة المرور (وزر Google جاهز للتفعيل)، صفحات دخول/إنشاء حساب/استعادة كلمة المرور، وحماية الصفحات الداخلية.

✅ إدارة المهام

عنوان، وصف، تاريخ استحقاق، أولوية (عالية/متوسطة/منخفضة)، حالة (انتظار/جارٍ/مكتمل)، وربط بمشروع.

🔁 مهام متكرّرة

مهام يومية أو أسبوعية تتولّد تلقائياً عند إكمالها.

📁 مشاريع ومسارات

اسم، مسار، لون، ونسبة إنجاز محسوبة تلقائياً مع شريط تقدّم.

📅 عروض متعددة

لوحة "اليوم"، عرض أسبوعي على شكل تقويم 7 أيام، كل المهام مع فلاتر، والمشاريع.

🌍 ثنائي اللغة

عربي (RTL) وإنجليزي (LTR) مع زر تبديل يحفظ تفضيلك، وتبديل اتجاه تلقائي.

🌗 فاتح / داكن

وضع فاتح ودافئ + وضع داكن، بتصميم مينمال نظيف على طراز Linear/Notion.

📧 ملخّص يومي بالإيميل

دالة مجدولة عبر pg_cron ترسل صباحاً ملخّص مهام اليوم لكل مستخدم.

المشاكل / الأمور التي تحتاج حلاً

1. حدّ إرسال الإيميل في Supabase أولوية عالية

خدمة الإيميل المدمجة في Supabase مخصّصة للتجربة ومحدودة بعدد قليل من الرسائل بالساعة، ما يمنع المستخدمين الجدد من إكمال التسجيل (ويعطّل إيميلات التأكيد والاستعادة).

الحل: إعداد SMTP مخصّص عبر Resend (يتطلّب توثيق دومين + مفتاح API). يزيل الحدّ نهائياً ويُشغّل كل إيميلات المصادقة والملخّص اليومي. مؤقتاً تم تعطيل "تأكيد الإيميل" للسماح بالدخول الفوري.

2. مفتاح Resend للملخّص اليومي أولوية متوسطة

دالة الملخّص اليومي منشورة والكرون نشِط، لكنها تحتاج مفتاح API لإرسال الرسائل فعلياً.

الحل: إضافة RESEND_API_KEY في أسرار الـ Edge Function داخل لوحة Supabase (واختيارياً RESEND_FROM و CRON_SECRET).

3. تفعيل تسجيل الدخول عبر Google أولوية متوسطة

زر "المتابعة عبر Google" موجود لكن المزوّد غير مفعّل، فيظهر خطأ "provider is not enabled".

الحل: إنشاء بيانات OAuth في Google Cloud وإضافة رابط https://lrrwsdesnlbzuzuaqjhr.supabase.co/auth/v1/callback، ثم تفعيل Google في Supabase ولصق Client ID/Secret.

4. توقيت لوحة "اليوم" أولوية منخفضة

لوحة "اليوم" تحسب التاريخ بتوقيت الخادم (UTC) بدل توقيتك المحلي، فقد تظهر مهمة "اليوم" متأخّرة بضع ساعات (تبقى ظاهرة في "كل المهام" و"الأسبوع").

الحل: احتساب "اليوم" بناءً على توقيت المتصفّح المحلي بدل الخادم.

5. تغيير كلمة المرور من الإعدادات تحسين

صفحة الإعدادات لا تحتوي حالياً على خيار لتغيير كلمة المرور — مهم لأن الحساب الأول أُنشئ بكلمة مرور مؤقتة.

الحل: إضافة حقل "تغيير كلمة المرور" في صفحة الإعدادات.

الخطوات القادمة المقترحة
  • توثيق دومين في Resend وضبط SMTP المخصّص في Supabase (يحلّ مشكلتَي الإيميل دفعة واحدة).
  • إعادة تفعيل "Confirm email" بعد ضبط SMTP لأمان أعلى.
  • تفعيل Google OAuth إن رغبت بتسجيل أسرع.
  • إضافة "تغيير كلمة المرور" وتصحيح توقيت لوحة "اليوم".
  • ربط Vercel بـ GitHub للنشر التلقائي عند كل تحديث.
